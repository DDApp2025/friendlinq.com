## About Friendlinq

Friendlinq (friendlinq.com) is a social media platform designed for adults who value simplicity, safety, and real connection. It is owned and operated by Albert (Robert Smith) through Infotech Inc. / Spire Group Inc.

### Core Features & Services
- Social networking and friend connections
- Messaging and communication tools (text, voice, video via Agora WebRTC)
- Interest-based community groups (gardening, travel, book clubs, health & wellness, etc.)
- Event discovery and local activity listings
- Photo and media sharing with simplified tools
- Profile creation and friend management
- Live streaming (host or join via Agora)
- Marketplace for community listings
- Simplified, intuitive user interface with large text and clear navigation
- Enhanced privacy and safety features
- Free to join and use — no subscription fees for core features
- Stripe-powered subscription for optional dating/premium mode

### Target Audiences
- Adults who want a simpler, safer social network
- People who find Facebook, Instagram, and other platforms too complex or invasive
- Family members who want to stay connected across generations
- Community organizations looking for digital social tools

### Value Propositions
- Purpose-built for simplicity (not a general platform retrofitted)
- Intuitive interface designed for comfort, not complexity
- Safety-focused with privacy protections
- No ads, no data selling, no algorithm manipulation
- Bridges generational gaps — family members of any age can join

## Technical Stack

- **Frontend:** React 19 (Create React App, react-scripts 5.0.1)
- **Router:** react-router-dom 7.13 — BrowserRouter (path-based)
- **State:** Redux + Redux Persist + Redux Thunk
- **Frontend hosting:** GitHub Pages via repo DDApp2025/friendlinq.com
- **Backend APIs (DO NOT MODIFY):**
  - Node.js: https://natural.friendlinq.com (posts, chat, portfolio, friends, notifications)
  - .NET: https://unpokedfolks.com/api (auth, groups, marketplace, payments, settings)
- **Real-time:** Socket.IO (messaging), Agora WebRTC (video/audio calls, live streams)
- **Payments:** Stripe (subscription plans)
- **Build:** `npm run build` → `/build/` directory
- **Deploy:** `npm run deploy` → runs `gh-pages -d build` → pushes to gh-pages branch
- **Domain:** friendlinq.com (points to GitHub Pages)
- **Local project:** `C:\Projects\friendlinq.com`
